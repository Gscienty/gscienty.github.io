1.通过文件的扩展名来判断
这是一种最鸡肋的方法。

2.通过C#自身提供的方法进行判断
bool isJPG = false; System.Drawing.Image img = System.Drawing.Image.FromFile(filesName); if ( img.RawFormat.Equals(System.Drawing.Imaging.ImageFormat.Jpeg) ) { isJPG = true; }

3.通过读取文件内容来判断
JPEG格式：以0xFFD8FF开头。
BMP格式：以0x424D开头。
PNG格式：以0x89504E47开头。
GIF格式：以0x47494638开头。
