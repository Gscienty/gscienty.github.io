Import Example Dataset

mongoDB的连接字符串格式如下：mongodb://host:27017

根据Docs的Guide，首先向MongoDB导入我们一个外部的数据集。
Docs上提供给我们一个现有的庞大的数据集，地址为： https://raw.githubusercontent.com/mongodb/docs-assets/primer-dataset/dataset.json
将该数据集保存到本地（本机测试目录为：D:\MongoDB\examples\），命名为"primer-dataset.json" (本身机器渣，Atom近乎崩溃，毕竟11MB左右)。

将数据填充到MongoDB当中。利用系统Shell或cmd，通过 mongoimport将文档添加到restaurants集合当中（mongoimport.exe在MongoDB bin目录下，可以通过设置系统变量节省“cd”时间。）
如果这个集合已经存在于test数据库中，那这个操作将会首先删除restaurants集合。 

        mongoimport --db test --collection restaurants --drop --file D:\MongoDB\examples\primer-dataset.json

命令行显示 导入25359条文档。 mongoimport将连接到运行监听在27017端口上的mongod实例。如果mongod运行在其他终端或端口上时，需要添加--host选项指定终端 --port选项指定端口。

在mongo中使用db.restaurants.findOne()命令查看，显示成功导入。

如果你在启动着MongoDB的mongod时，在浏览器中浏览http://localhost:27017，将会返回“It looks like you are trying to access MongoDB over HTTP on the native driver port.”提示（算彩蛋吗？）
