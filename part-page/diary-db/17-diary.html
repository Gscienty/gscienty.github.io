Find or Query / Update / Remove Data with C# Driver
我们尽可能想象内存是透明的，我们直接操作的是Document。

MongoDB C# Driver提供了几种Find方法（1.10）
建议使用MongoDB.Driver.Builder.Query()来构造约束条件。

  1. FindAllAs    获取Collection下的所有Documents
  2. FindAndModify    查询并将查询到的Documents进行修改
  3. FindAndRemove    查询并将查询到的Documents删除
  4. FindAs    根据约束条件查询Documents
  5. FindOneAs    根据约束条件查询Documents，并将第一个Document返回
  6. FindOneByIdAs    根据ID获取Document

这里还是着重将Entity修改/查询，而非BSON。

Save a Document
你可以通过Save方法修改一个已经存在的文档 

        entity.Name = "Dick";
        collection.Save(entity);

Update an Existing Document
与Save不同的是，Save是发送一个曾从Server发来的一个完整的Entity。而Update仅仅发送的是修改的部分。 

        var query = Query.EQ(e=>e.Id, id);
        var update = Update.Set(e=>e.Name, "Herry");
        collection.Update(query, update);

Remove an Existing Document 

        var query = Query.EQ(e=>e.Id, id);
        collection.Remove(query);
